## Kisan Setu AI – Frontend (Farmer Console)

This is a minimal **Next.js 14 + TypeScript + Tailwind** app that serves as the visual shell for the Agri-OS Farmer Console.

### Getting started

```bash
cd frontend
npm install
npm run dev
```

Then open `http://localhost:3000` to view the console.

### Current features

- Dark, green-accented theme inspired by fields and soil.
- Bilingual hero content (Marathi + Hindi) describing the voice-first experience.
- Placeholder call-to-action section for future WhatsApp voice/image flows.

As the backend and Bedrock/RAG agents evolve, this console will connect to APIs for:
- Uploading / capturing voice and images.
- Viewing personalized advice and mandi forecasts.
- Managing farmer profiles and soil/“sarkari-mitra” documents.
